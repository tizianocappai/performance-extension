What's measured
Everything your page is doing,
at a glance

JS Heap Memory

Live used / allocated / limit values with a progress bar. Alerts fire when you exceed 500 MB.

Frame Rate

Measured via requestAnimationFrame. Color-coded: green ≥ 50, orange ≥ 30, red below.

CPU Usage

System-wide CPU percentage sampled every second via the chrome.system.cpu API.

Long Tasks

Counts every main-thread block over 50 ms — the root cause of janky, unresponsive UIs.

30-second Charts

Sparkline graphs for FPS, RAM, and CPU show trends over time, not just the current snapshot.

Always-visible Overlay

A draggable widget injected into every page keeps metrics visible while you interact with the site — no need to keep the popup open.

Core Web Vitals

FCP, LCP, CLS, and INP measured live with Google threshold color coding — the same signals used for Search ranking.

Network Conditions

Estimated downlink speed (Mbps), RTT, and connection type (4g/3g/…) via navigator.connection.

System & Process RAM

Total and available system memory via chrome.system.memory, plus the private RAM of the tab's renderer process.

Export Snapshot

Download a self-contained HTML performance report with charts, avg/min/max stats, long task log, and 30-second raw history.

100% Local

All data lives in chrome.storage.session. Nothing is sent anywhere, ever.
